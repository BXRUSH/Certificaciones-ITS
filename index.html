    Certificaion

    Las certificaciones hoy en día son importantes para la industria, ya que estas garantizan una conocimiento adquirido y que en algunos casos es necesario para obtener el empleo que se desea conseguir, por eso en esta presentación hablaremos sobre lo que es una certificación y las que creemos que son las mas importantes que un Ingeniero en Tecnología de Software debe de tener.

    ¿Qué es una certificación?

    En internet podemos encontrar diversas definiciones y como tal todas ellas tienen el mismo fin. Como por ejemplo: “Las certificaciónes el procedimiento mediante el cual un organismo autorizado valida ciertos aspectos (calidad, fidelidad, etc.) de una organización, institución o persona.” (Méndez, 2018) Las certificaciones pueden ser desde habilidades manuales como digitales.

    ¿Para que sirve una certificación?

    Las certificaciones tienen como propósito validar nuestras habilidades en las tecnologías de información, estás nos ayudan a fortalecer nuestra hoja de vida al momento de buscar alguno empleo. Las personas que obtienen una certificación relacionada con su empleo presentan un aumento a su salario, ya que están mas preparadas académicamente que el resto de trabajadores.

    Microsoft Certified Application Developer (MCAD)

    Esta dirigida a Analistas, Programadores, Ingenieros de Software, Especialistas en aplicaciones y desarrolladores de software, quienes diseñan e implementan aplicaciones multicapa avanzadas utilizando Web Services de Microsoft .NET Studios.

    Que es un lenguaje de programacion

    Un lenguaje de programación esta destinado a desarrollar aplicaciones para la web y crear páginas web, favoreciendo la conexión entre los servidores y la interfaz de usuario.

    JAVASCRIPT

    En el área de nuestra ingeniería hay muchos tipos de certificaciones que podemos sacar, ya sea con cursos en línea o presenciales, y JavaScript es uno de los más importantes que podemos tener ya que es junto con Python, uno de los mas prescindibles a la hora de hablar de certificados para nuestra ingeniería, pero… ¿Qué es JavaScript y qué lo hace tan importante? JavaScript es un lenguaje de programación dinámico que tiene la capacidad de ser utilizado en muchos dispositivos diferentes. Puede usarse en ordenadores personales, servidores web y teléfonos inteligentes. Es un lenguaje interpretado, orientado a objetos, débilmente tipado y dinámico

    PYTHON

    Python para los ingenieros en Software/Sistemas y en general en la programación es probablemente la herramienta de programación más utilizada generalmente, se sabe que es bastante sencilla de usar debido a su gran similitud al lenguaje humano, por lo tanto, saber utilizarla y comprenderla tiene beneficios y ventajas que ninguna otra herramienta de programación tiene, así que para poder comprenderla nosotros. Python es un lenguaje de programación ampliamente utilizado en las aplicaciones web, el desarrollo de software, la ciencia de datos y el machine learning (ML). Los desarrolladores utilizan Python porque es eficiente y fácil de aprender, además de que se puede ejecutar en muchas plataformas diferentes. El software Python se puede descargar gratis, se integra bien a todos los tipos de sistemas y aumenta la velocidad del desarrollo.

    Microsoft Certified Solution Developer (MCSD)

    Esta certificación valida las habilidades de una persona para diseñar y construir aplicaciones en entorno de tecnología Microsoft. Posee tres rutas de certificación, Aplicaciones (Apps) de Windows Store, Aplicaciones Web y Gestión de Ciclo de Vida de Aplicaciones. Para obtenerla deben presentarse hasta 6 exámenes

    Microsoft Certified Application Developer (MCAD)

    Esta dirigida a Analistas, Programadores, Ingenieros de Software, Especialistas en aplicaciones y desarrolladores de software, quienes diseñan e implementan aplicaciones multicapa avanzadas utilizando Web Services de Microsoft .NET Studios.
